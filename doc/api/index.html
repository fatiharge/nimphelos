Nimphelos Package

Nimphelos is a class that provides methods for managing service calls and handling exceptions. It can handle common errors like Dio and JSON errors. Additionally, it offers a flexible structure to add custom exception handlers. The package is used to facilitate error handling and network client operations in Flutter applications.

Usage

NimphelosModel res = await nimphelos.fetchData<String>(
  response: () async {
    return (await dio.get("/welcome/test200")).data;
  },
);

Custom Hata Mesajları

Eğer ki alert mesajını beğenmediyseniz bu mesajları uygulamasınıza göre değiştirebilirsiniz. Örneğin badCertificate hatası alıyorsanız aşağıdaki ekleme yapmanız yeterli olacaktır.

NimphelosModel res = await nimphelos.fetchData<String>(
  response: () async {
    return (await dio.get("/welcome/test200")).data;
  },
  context: context,
  activeAlert: true,
  exceptionDioHandlers: DioExceptionHandlerModel(
    badCertificateModel:  ExceptionExplanationModel("Custom title","Custom content","Custom button"),
  ),
);

Custom Error Action

Eğer ki hata durumunda daha karmaşık işlemler yapmanız gerekiyorsa kendiniz de fonksiyon ekleyebilrisniz. Örneğin connectionError hatası için consola yazı yazdırmak istiyorsanız aşağıdaki eklmeyi yapmanız yeterlidir. default olan alert kodu çalışmayacaktır. Sizin yazdığınız kod çalışır.

NimphelosModel res = await nimphelos.fetchData<String>(
  response: () async {
    return (await dio.get("/welcome/test200")).data;
  },
  context: context,
  activeAlert: true,
  exceptionDioHandlers: DioExceptionHandlerModel(
    badCertificateModel:  ExceptionExplanationModel("Custom title","Custom content","Custom button"),
    connectionError: (exception) {
        print("Custom action");
    },
  ),
);

willAlwaysWork Action

Eğer ki ui işlerinden farklı işlemler yapmanız gerekiyorsa. Mesela hata durumunda loglama gibi bu durumlardan bağımsız ilgili Exception'a ulaşmak isteyebilirsiniz. Aşağıdaki örnek kullanımı görebilirsiniz. activeWillAlwaysWork parametresi true olmalıdır. Default'u true dur.

NimphelosModel res = await nimphelos.fetchData<String>(
  response: () async {
    return (await dio.get("/welcome/test200")).data;
  },
  context: context,
  activeAlert: true,
  exceptionDioHandlers: DioExceptionHandlerModel(
    badCertificateModel:  ExceptionExplanationModel("Custom title","Custom content","Custom button"),
    connectionError: (exception) {
        print("Custom action");
    },
    willAlwaysWork: <E>(exception) {
       print("willAlwaysWork....");
    },
  ),
);

General Function

Eğer ki tüm hatalarda aynı aksiyonu almak istiyorsanız. context'i eklemeyin ve generalFunction'u kullananın.

NimphelosModel res = await nimphelos.fetchData<String>(
  response: () async {
    return (await dio.get("/welcome/test200")).data;
  },
  activeAlert: true,
  generalFunction: <E>(exception) {
    print("generalFunction");
    return true;
  },
);
